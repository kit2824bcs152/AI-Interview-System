# AI Interview Simulator with Emotional Analysis

A lightweight browser-based interview practice app that asks realistic questions, captures your answers, and provides instant feedback on:

- sentiment
- confidence
- emotional tone
- coaching tips

## Run locally

Open [index.html](index.html) in your browser, or serve the folder with a simple static server.

Example:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000
